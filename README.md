Projeto Árvore Binária

# Visão Geral
Este projeto implementa uma Árvore Binária simples em Python, demonstrando as principais operações e a visualização da estrutura de dados.
O objetivo principal é entender e visualizar como os dados são armazenados e manipulados em uma estrutura de árvore.

# Requisitos do Sistema
Para rodar este projeto, você precisa ter o Python instalado (versão 3.x).

# Instalação de Dependências
Este projeto depende da biblioteca matplotlib para a visualização.

Você pode instalar todas as dependências listadas no arquivo requirements.txt usando o pip:
pip install -r requirements.txt
